== Installation ==
 
This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. use shortcode in page,post, widgets.
4. you can use news ticker in your theme by using '<?php echo do_shortcode('YOUR SHORTCODE');?>'

Shortcorde
1. General ticker
<pre>[ticker_list id="1"]</pre>

2. speed change
<pre>[ticker_list id="2" speed="1000" ] </pre>

3.typespeed change
<pre>[ticker_list id="3"  typespeed="30" ]</pre>
4. text change
<pre>[ticker_list id="4"  text="News"]</pre>

5. color variation

<pre>[ticker_list id="2" color="green"]</pre>
